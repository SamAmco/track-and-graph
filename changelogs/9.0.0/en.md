**Version 9.0.0** introduces three new reminder types to help you stay on top of your tracking:

* **Periodic Reminders**: Set reminders that repeat at regular intervals (e.g., every 3 days, every 2 weeks).
* **Month Day Reminders**: Set reminders for specific days of the month (e.g. the first Monday or last Friday of each month).
* **Time Since Last**: Set reminders based on the time elapsed since you last tracked a specific item. (These also work with functions provided your function only produces data points when you track something).
