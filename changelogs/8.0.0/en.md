**Version 8.0.0** introduces Functions, a powerful new way to create custom data sources by transforming and combining existing trackers using a visual node editor.

![Functions Screenshot](https://raw.githubusercontent.com/SamAmco/track-and-graph/refs/heads/master/changelogs/8.0.0/functions_screenshot.png)

To get started, create a new Function in any group by tapping the **+** icon and selecting "Function".

![Create Function](https://raw.githubusercontent.com/SamAmco/track-and-graph/refs/heads/master/changelogs/8.0.0/create_function_en.png)

For more information [check out the tutorial here](https://samamco.github.io/track-and-graph/tutorial/functions/)
