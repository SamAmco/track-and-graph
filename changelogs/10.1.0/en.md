## Search! 🎉

**Version 10.1.0 introduces Search!**

### Find components faster

Use the new search icon in the top bar to find components inside the current group hierarchy.

Search is fuzzy, so close matches work, and descriptions are included too.

Import and export now share one compact tabbed dialog, and each group remembers the last tab you used.

### Search history

Tracker and Function history screens also have search. Data points are filtered by matching value, description, or notes but kept in reverse chronological order.

### Search notes

The notes screen now supports search, plus filters for global and data point notes.
